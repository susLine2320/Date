# 日付表示プラグイン
このプラグインは日付情報を設定するプラグイン。

### panelインデックス
218…年
219…月
220…日
221…曜日（日1～土7）

### soundインデックス
なし

### beaconインデックス
607…任意年・曜日設定（設定年号*10+設定曜日:日1～土7）
#### （注意）
10以下（曜日のみ）設定…年は現在年が設定される。

607番地上子を置くと指定の年・曜日で現在日付から近い日が自動で設定される。

607番地上子を置かない・あるいは任意番号を0とすると現在の日付が設定される。

日付変更は未検証であることに留意すること。

### signalインデックス
なし

### SWインデックス
なし

## 著作権
何かしらのライセンスを設定しているわけではないが、単体での再配布は不可。
車両データに同封する場合、事前連絡不要。なお当方の把握のため、事後に必ず連絡すること。

## その他
このプラグインにはソースコードを同梱していない。そのためパネルインデックス番号を変更希望の者は下記まで連絡すること。
